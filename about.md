# Rod's Fangame Tools

This mod adds some cool <cg>utilities</c> for anyone who wishes to make <cy>Geometry Dash fangames</c> or just wants to mess around with level data.

## Features

- Level Utilities
  - Copy Main Levels
  - Copy Level String
  - Paste Level String
  - Encoded/Decoded Level Type
- Miscellaneous
  - Preview Main Level Data

## Recommended Mods

**Unlisted Objects in Editor**: This mod adds useful objects in the editor (most importantly, the secret coin).

<mod:legowiifun.unlisted_objects_in_editor>

**More Icons**: This mod lets you create and test new icons which you can add to your fangames.

## Incompatibilities

- Eclipse: Read-only level preview code may overlap a bit, but it seems to work fine.
